# Chenyu Hu Personal Website

A premium minimalist personal portfolio built with plain HTML, CSS, and JavaScript. The page is designed for an academic, research, engineering, or early-career professional profile.

## Structure

```text
index.html        # Page content and section structure
css/style.css     # Complete visual system and responsive layout
js/main.js        # Mobile navigation, scroll state, reveal animations
assets/           # Profile photo, resume PDF, and supporting files
```

## Sections

- Hero
- About Me
- Education
- Projects
- Experience
- Skills
- Contact

## Customize

Edit `index.html` to update your name, tagline, biography, projects, experience, education, skills, and contact links.

## Design

The style is controlled by CSS variables at the top of `css/style.css`.

Key choices:

- Editorial display typography with a clean sans-serif body
- Neutral white, black, soft gray, and muted olive-gray accent colors
- Thin section dividers and restrained cards
- Responsive grids for desktop, tablet, and mobile
- Subtle hover states and scroll reveal animation

## Local Preview

Open `index.html` directly in a browser. No build step or development server is required.
